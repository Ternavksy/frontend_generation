# SegLabel AI Frontend

React + TypeScript + Tailwind frontend для системы автоматической разметки изображений.

## Установить

```bash
npm install
```

## Запустить

```bash
npm run dev
```

## Структура

- `src/App.tsx` — маршрутизация страниц
- `src/pages` — страницы: авторизация, дашборд, рабочая область, загрузка, модели, профиль, админка
- `src/components` — общие UI-компоненты
- `src/styles` — Tailwind CSS

## Особенности

- Реализован интерфейс по ТЗ: авторизация, загрузка датасетов, рабочая область CVAT-подобной архитектуры, управление моделями, личный кабинет, админка
- Поддержка темной темы и адаптивной компоновки
- Настроен Vite + TypeScript + Tailwind
